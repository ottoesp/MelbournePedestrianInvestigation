# Melbourne Pedestrian Investigation

This is a statistical investigation based on the City of Melbourne pedestrian counting system. I am trying to isolate
the effect of the pandemic period on pedestrian traffic and quantify our changing trends in the city (such as remote learning and working from home). 

As an undergraduate maths student this has really extended my knowledge from uni, involving some new statistical techniques such as Linear Mixed Models. 

Main branch is hosted [here](https://ottoesperson.com), feel free to check out my work :)
